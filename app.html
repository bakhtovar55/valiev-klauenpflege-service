VALIEV KLAUENPFLEGE – APP

Interner Arbeitsmodus für Bakhtovar (Tablet / Smartphone)

1. Kuh hinzufügen / bearbeiten

Neue Kühe anlegen oder bestehende bearbeiten. Gleiche Kuh-Nummer = Aktualisierung.

2. Behandlung eintragen

Behandlung mit Diagnose, Fuß, Block, Verband, Produkt und nächster Kontrolle dokumentieren.

3. Kuh-Historie anzeigen

Alle Behandlungen einer bestimmten Kuh sehen.

Datum Diagnose Fuß Block Verband Produkt Bemerkung
4. Liste aller Kühe

Übersicht über alle angelegten Kühe.

Kuh Hof Adresse Geburt Lahm? Notiz
5. Tagesübersicht / PDF-Bericht

Alle Behandlungen eines Tages anzeigen und für PDF / Druck vorbereiten.

Hof Kuh Datum Diagnose Fuß Block Verband Produkt Nächste Kontrolle
6. Datenverwaltung / Backup

Alle Daten werden lokal im Browser gespeichert (localStorage). Kein Internet, kein Server – nur auf diesem Gerät. Backup hilft beim Gerätewechsel.
